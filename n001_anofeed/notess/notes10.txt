This is the most important step.

NextAuth receives:

{
  "identifier": "john",
  "password": "secret123"
}

Then NextAuth:

Finds user in database.
Compares password.
Creates session if valid.
Returns result.
Why redirect ?

Without:

redirect: false

NextAuth automatically redirects.

With:

redirect: false

We control everything ourselves.

We can:

Show custom toast
Handle errors
Redirect manually
